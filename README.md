# 📈 RockyBot: News Research Tool

RockyBot is an AI-powered research assistant that allows users to extract, analyze, and query information from multiple news article URLs in real time. Built using LangChain, Google Gemini API, and FAISS, it enables fast and accurate question answering based on live web content.

## 🚀 Features

- 🔗 Extracts content from multiple URLs
- ✂️ Splits and processes large text efficiently
- 🧠 Generates embeddings using Google Gemini
- 📦 Stores data using FAISS vector database
- 💬 Ask questions and get context-based answers
- 📚 Provides source-aware responses
- ⚡ Fast and interactive UI with Streamlit

## 🛠️ Tech Stack

- Python
- LangChain
- Google Gemini API
- FAISS (Vector Database)
- Streamlit
- Unstructured (for web scraping)

## 🧠 How It Works

1. User inputs one or more news article URLs  
2. Content is extracted and split into chunks  
3. Text is converted into embeddings  
4. Stored in FAISS vector database  
5. User asks a question  
6. Relevant context is retrieved  
7. Gemini generates an accurate answer  

## 📸 Use Case

- News analysis  
- Research assistance  
- Content summarization  
- Multi-source information retrieval
